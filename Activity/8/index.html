Write a function that takes a list, counts the number of nodes, and returns it.

size_t size( sNode* ) ;

Write a function that takes a list of elem_ts and an elem_t, t, counts the number of occurrences of the target elem_t, returns this value.

size_t count( sNode*, elem_t t ) ;

Test it. Get it right. Maybe implement the list print function, from the notes.

Write a function that takes a list of elem_ts and an elem_t, removes the first node that contains the elem_t. Return a pointer to the beginning of the list. No memory leaks.

sNode* remove_first( sNode *, elem_t ) ;

Test it. Get it right. Maybe implement the list print function, from the notes.

Write a function that takes a list of elem_ts and an elem_t, removes every node that contains the elem_t. Return a pointer to the beginning of the list. No memory leaks.

sNode* remove_all( sNode *, elem_t ) ;
