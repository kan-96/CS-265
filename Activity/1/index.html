You may work in groups, explain to each other what's happening. Place the username of each group member at the top of the file, one per line, yours first, followed by a blank line.

Files for this activity can be found in ~kschmidt/public_html/CS265/Activities/1/ . Copy them as needed.

Copying Files

Let's copy some files over, for practise.

  1. In lecture we discussed paths, to identify files in the filesystem.
  2. So, the path to this HTML file on tux is:
    /home/kschmidt/public_html/CS265/Activities/1/index.html
    or

  3. Recall that . is a relative path which refers to the current directory
  4. If you are not in your Activity 1 directory, navigate there now

    cp ~kschmidt/public_html/CS265/Activities/1/index.html .
